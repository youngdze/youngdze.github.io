二叉树的插入规则：

分层遍历广度查询(Breadth First Search)

分层遍历相对复杂，它是一层一层进行遍历的，图示(来自 Wikipedia)：

具体思路是先定义一个数组，然后把各层节点在每次循环压入这个数组，每次循环都将这一层节点的值打印出来，同时设置循环结束条件为当前结点为最后节点。

查找

查找其实相对简单，毕竟查找数值与当前结点，小则循环到左节点，大则循环到右节点，等于则返回，查找不到返回 null。
